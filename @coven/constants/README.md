## Constants

- `EMPTY_ARRAY`: A frozen empty array (`[]`).
- `EMPTY_OBJECT`: A frozen empty plain object (`{}`).
- `EMPTY_STRING`: An empty string (`""`).

## Example

```typescript
import { EMPTY_ARRAY } from "@coven/constants";

console.log(EMPTY_ARRAY); // []
```
